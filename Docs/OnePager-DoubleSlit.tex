\textbf{Principle of Maximal Information Collapse.}
When an observer extracts which-path information from a quantum system, the residual coherence capacity is:
\[
\boxed{\;\text{Residual} = 1 - \frac{\text{MI}_{\text{extracted}}}{k_B T \ln 2} \;\in\; [0,\,1]\;}
\]
Extract 0 bits $\implies$ full interference. Extract 1 bit $\implies$ complete decoherence.\\[3pt]

\section{The Physical Argument}

\noindent A particle traverses a double-slit apparatus. Its spatial path qubit is a $2\times 2$ density matrix $\rho$.

\begin{enumerate}[leftmargin=*, itemsep=2pt]
\item \textbf{No detector.} Off-diagonals $\rho_{01} \ne 0$ carry coherence; interference fringes appear ($V > 0$).
\item \textbf{Which-path detector.} The L\"uders channel $\mathcal{E}(\rho) = \Pi_0\rho\Pi_0 + \Pi_1\rho\Pi_1$ zeros off-diagonals: $V \to 0$. Englert guarantees $I$ rises accordingly.
\item \textbf{Thermodynamic cost.} The observer's memory now holds a classical bit. Erasing it against a bath at $T$ dissipates $\ge k_B T \ln 2$ joules (Landauer). This energy is \emph{exactly} the cost of destroying coherence.
\item \textbf{Maximal collapse.} Normalizing: if the observer extracts the full 1-bit capacity, residual coherence $= 0$. Partial extraction leaves partial visibility. The bound is tight at both endpoints.
\end{enumerate}

\noindent \textbf{Conclusion.} The observer does not ``collapse reality'' by an act of consciousness. The observer pays an irreversible thermodynamic cost that is \emph{mathematically identical} to decoherence. \textbf{Measurement IS thermodynamics.}
